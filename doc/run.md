# How to run

Let $n$ be the question number

## C++

```shell
make bin/test_n
./bin/test_n
```

To compile and run all tests:

```shell
make run
```

## Python

`-s` is optional, it shows timing for each individual test.

```shell
pytest test/python/test_n.py -s
```

To compile and run all tests:

```shell
pytest test/python/ -s
```

## JavaScript

```shell
node test/js/test_n.js
```
